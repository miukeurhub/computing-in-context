  1. Create an empty list called num_level_4 where the numeric values of column Num Level 4 form the dataset will be stored.

  2. Open the CSV file in read mode (‘r) to make sure the dile will automatically close once the block of code is done and assigns it to the variable ‘data’.

  3. Read data as a dictionary where column headers become keys and the values in each row are accessible by column name.

  4. Loop through each row in the CSV file and check id the value in the column of Num Level 4 is a numeric string using isdigit(). If True, the value is converted to an integer and added to the list.

  5. Coverts the valid string in ‘Num Level 4’ to an integer and appends it to num_level_4

Mean 6. Calculate the total sum of all elements in num_level_4 and finds the number of elements in num_level_4. Next, divide the sum by count of elements to get the mean value.

Median 7. Create a sorted version of num_level_4 to find the median, and calculate the number of elements. 8. Checks if the number of elements is odd. If it is Odd, the median is the middle element, found at num_l4_sorted[n//2]. If it is Even, the median is the average of the two middle elements. So, we need to find the middle element 1 and middle element 2 and divided by 2.
